### Mail report wrapper

A simple wrapper to send the result of a command by mail.

## Usage

Environment variables:
* SMTP_HOST
* SMTP_PORT
* SMTP_USER
* SMTP_PASSWORD
* SMTP_PROXY
* MAIL_SUBJECT
* MAIL_SENDER
* MAIL_RECEIVERS
* MAIL_TEMPLATE_FILE
* MAIL_ATTACHMENTS_DIR

```
mail-report-wrapper command arg1 arg2
```
